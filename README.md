## 技术点：

* 底部bar：

1. 创建一个窗口
2. 拦截窗口的触摸事件传递

* 转场动画：

1. 自定义转场动画执行者
2. 通过绘制layer的mask 进行 base的核心动画
注意：在view显示和消失时移除和记录navgation的转场代理
